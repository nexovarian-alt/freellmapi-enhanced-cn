# FreeLLMAPI Enhanced CN v1.1.0 正式发布

FreeLLMAPI Enhanced CN v1.1.0 是面向中国普通用户、NAS 用户和 AI 工具玩家的稳定版本。它保持 FreeLLMAPI 原有使用体验，重点改善 Docker 环境下的海外 AI 网络连接、Provider 状态判断和故障恢复。

## 用户能感受到的变化

- 中国网络增强：Windows、Mac 和 NAS 的 Docker 部署可以连接用户已有的代理环境。
- 海外 Provider 更稳定：Google AI Studio、Groq、OpenRouter 可通过统一代理配置访问。
- Provider 智能诊断：分别显示网络、认证、额度和模型状态，减少把网络问题误认为 Key 失效的情况。
- 自动恢复：临时网络故障恢复后，Provider 可重新检测并恢复使用。
- 兼容升级：保留原有数据目录、Volume 和配置即可升级，无需重新添加 Provider。
- Docker 部署优化：改善健康检查和普通用户的安装、排障体验。
